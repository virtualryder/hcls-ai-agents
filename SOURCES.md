# Sources

All cited figures used in the GTM decks and docs live in
**[`gtm/HCLS-DECK-SOURCES.md`](gtm/HCLS-DECK-SOURCES.md)** — every stat is tagged by source class
(`[gov/peer-reviewed]` · `[industry-research]` · `[sector-press/estimate]` · `[vendor]` · `[modeled]`)
with a working URL, and vendor/modeled figures are flagged and never used as a lead claim.

Regulatory and security control sources are cited inline in `docs/NIST-800-53-CONTROL-MATRIX.md`,
`docs/OWASP-LLM-ATLAS-MAPPING.md`, and `docs/THREAT-MODEL.md`.
